# Screenshots

No screenshots are included in this initial release — this suite is a
set of markdown-based Claude Code skills with no GUI of its own, so
"screenshots" here would mean captures of a Claude Code session
running the pipeline (e.g. the orchestrator's progress output, or a
generated `REPOSITORY_SCORECARD.md` rendered in an editor).

If you'd like to contribute screenshots showing a real run of the
suite (with any sensitive repository details redacted), see
[`../../CONTRIBUTING.md`](../../CONTRIBUTING.md) — these are
genuinely useful for new users deciding whether to install the suite,
and a welcome first contribution.

Suggested screenshots, if you're contributing one:

- A Claude Code session invoking `repository-orchestrator`
- A generated `REPOSITORY_SCORECARD.md` rendered in an editor or
  GitHub's markdown preview
- A `WORKFLOW_STATE.md` mid-pipeline, showing stage progress
