# Diagrams

Mermaid source files for the four diagrams rendered in
[`../../docs/architecture.md`](../../docs/architecture.md) and (the
pipeline diagram only) in the root [`README.md`](../../README.md).
Kept here as standalone `.mmd` files so they can be rendered to PNG/SVG
independently (e.g. via the [Mermaid CLI](https://github.com/mermaid-js/mermaid-cli)
or the [Mermaid Live Editor](https://mermaid.live)) for use in slides,
issues, or anywhere that doesn't render Mermaid natively.

| File | Matches |
|---|---|
| `skill-pipeline.mmd` | "Skill Pipeline" diagram |
| `artifact-flow.mmd` | "Artifact Flow" diagram |
| `quality-gate-flow.mmd` | "Quality Gate Flow" diagram |
| `repository-lifecycle.mmd` | "Repository Lifecycle" diagram |

If you update a diagram in `docs/architecture.md`, update the matching
`.mmd` file here too — they're meant to stay identical.
